Standard prefixes for functions, types, variables, etc...  This is to help avoid collisions, as things within audio DSP programming tend to get passed all over
the place.

va_ == variable (not static or constant, nor within the scope of a struct or class)
sv_ == static variable
co_ == constant
st_ == struct
t_  == type
g_  == function returning an instance of a type we've defined in the library  
a_  == argument variables passed to a function
f_  == variables declared within the scope of a function
v_  == function returning void
f_  == function returning float
d_  == function returning double
fp_ == function pointer
si_ == signal
sl_ == slot
